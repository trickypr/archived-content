The PromiseUtils.jsm JavaScript code module offers some useful functions related to . To use it, you first need to import the code module into your JavaScript scope:

Components.utils.import("resource://gre/modules/PromiseUtils.jsm");

Methods

defer()

Creates a new pending and provides methods to resolve or reject this .

This method was previously implemented as Promise.defer(), which is obsolete since Gecko 30. And PromiseUtils.defer uses instead of Promise.jsm's Promise.

Deferred defer();
Parameters

None.

Return value

A new object, containing the new promise in the promise property, and the methods to change its state in the resolve and reject properties. See the Deferred documentation for details.
