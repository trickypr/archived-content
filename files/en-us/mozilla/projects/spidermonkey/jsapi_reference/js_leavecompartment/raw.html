Leave a the compartment, returning to the compartment active before the corresponding JS_EnterCompartment.

Syntax

void
JS_LeaveCompartment(JSContext *cx, JSCompartment *oldCompartment);
Name Type Description
cx --> The context in which to leave the compartment.
oldCompartment --> Value returned by previous call to -->.

Description

Every --> has a current compartment. Any access to an object in a different compartment must be bracketed by calls to --> and JS_LeaveCompartment.
