Description

JSID_VOID does not occur in JS scripts but may be used to indicate the absence of a valid . A void is not a valid id and only arises as an exceptional API return value, such as in . Embeddings must not pass JSID_VOID into JSAPI entry points expecting a jsid and do not need to handle JSID_VOID in hooks receiving a jsid except when explicitly noted in the API contract.

JSID_VOIDHANDLE is the handle to the which value is JSID_VOID.
