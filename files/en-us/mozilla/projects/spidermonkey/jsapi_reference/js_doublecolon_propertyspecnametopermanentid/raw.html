Create a that does not need to be marked for GC.

Syntax

bool
JS::PropertySpecNameToPermanentId(JSContext *cx, const char *name, jsid *idp);
Name Type Description
cx Pointer to a JS context from which to derive runtime information.
name const char * or .
idp Out parameter. Receives the created id.

Description

JS::PropertySpecNameToPermanentId creates a that does not need to be marked for GC from or . The resulting , on success, is either an interned string or a well-known symbol; either way it is immune to GC so there is no need to visit *idp during GC marking..
