Leave a local root scope, transferring the result value to the next enclosing root scope.

Syntax

void
JS_LeaveLocalRootScopeWithResult(JSContext *cx, jsval rval);
Name Type Description
cx --> Pointer to the context. This must be the same context that was passed to -->. -->
rval --> The result value that should remain protected from garbage collection.

Description

See --> for an explanation of local root scopes.

The protection this function affords to rval is tenuous. If there is an enclosing local root scope, then rval is transferred to it. Otherwise, the value is stored in an internal per-JSContext slot. This slot is rooted, but the value will eventually be overwritten by some other operation, and it is very difficult to figure out exactly when this will happenâ€”or more to the point, guarantee that it won't happen in the time it takes some specific chunk of code to run.

In any case, all other newborn values protected by the local root scope, aside from rval, become subject to garbage collection.
