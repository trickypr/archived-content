Frees the stack entry allocated by .

Syntax

void
JS_PopArguments(JSContext *cx, void *mark);
Name Type Description
cx Pointer to a JS context from which to derive runtime information.
mark void * Pointer to a void * which holds the stack frame pointer previously supplied by .

Description

JS_PopArguments frees the stack frame pointer previously allocated by and unroots the s which have been associated with it (those returned by JS_PushArguments as well).
