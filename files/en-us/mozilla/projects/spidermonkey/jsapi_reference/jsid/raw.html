A jsid is a JavaScript identifier for a property or method of an object.

jsid is either a 31-bit signed integer, interned string or object.

A few JSAPI functions use jsids instead of s for property names: , , and all functions with names ending in ById.

Also, there is an additional jsid value, , which does not occur in JS scripts but may be used to indicate the absence of a valid jsid. A void jsid is not a valid id and only arises as an exceptional API return value, such as in . Embeddings must not pass into JSAPI entry points expecting a jsid and do not need to handle in hooks receiving a jsid except when explicitly noted in the API contract.

A jsid is not implicitly convertible to or from a ; or must be used instead.
