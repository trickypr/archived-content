Flags

Name Description --> constructor flag
JSREG_FOLD Fold uppercase to lowercase. i

JSREG_STICKY Only match starting at lastIndex. y

Description

JS_NewRegExpObject and JS_NewUCRegExpObject create a new --> instance. The flags from the built-in --> constructor properties ignoreCase, global, multiline, and sticky are OR'd in with the provided flags parameter.

JS_NewRegExpObjectNoStatics and JS_NewUCRegExpObjectNoStatics create a new --> instance. They do not OR in any --> constructor properties.
