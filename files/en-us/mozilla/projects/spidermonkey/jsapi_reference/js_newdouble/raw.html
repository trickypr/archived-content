Allocates a copy of on the GC heap.

Syntax

jsdouble *
JS_NewDouble(JSContext *cx, jsdouble d);
Name Type Description
cx The context in which to create the new number.
d The number to copy.

Description

JS_NewDouble allocates a copy of the number d on the JavaScript garbage collection heap. This function is not very useful. To convert a jsdouble to a , use instead.

The new jsdouble is subject to garbage collection until you protect it using a local root, an object property, or the function.

On success, JS_NewDouble returns a pointer to the new number. Otherwise it returns NULL.

The application must treat the returned pointer as read-only. Modifying a jsdouble allocated with this function can cause undefined behavior.
