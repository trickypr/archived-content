Leave a local root scope.

Syntax

void
JS_LeaveLocalRootScope(JSContext *cx);
Name Type Description
cx Pointer to the context. This must be the same context passed to ().

Description

See for an explanation of local root scopes.

Leaving a nested local root scope causes all objects created since the matching JS_EnterLocalRootScope call to become subject to garbage collection. That is, they are not transferred to the enclosing local root scope. However, provides a way to transfer one value to the enclosing local root scope.
