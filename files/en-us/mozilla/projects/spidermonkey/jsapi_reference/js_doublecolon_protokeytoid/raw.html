Convert a to JS ID.

Syntax

void
JS::ProtoKeyToId(JSContext *cx, JSProtoKey key, JS::MutableHandleId idp);
Name Type Description
cx Pointer to a JS context from which to derive runtime information.
key The prototype key to convert.
vp Out parameter. *idp receives the converted id.

Description

JS::ProtoKeyToId converts a specified JS prototype key key, to a JS ID. JS::ProtoKeyToId stores a to *idp.
