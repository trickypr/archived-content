Returns a string representation of a JSVersion value.

Syntax

const char *
JS_VersionToString(JSVersion version);
Name Type Description
version Version value to convert.

Description

JS_VersionToString attempts to convert the version to a const char * string representation. JS_VersionToString may return any of the following values:

Enumeration String
JSVERSION_1_0 "1.0"
JSVERSION_1_1 "1.1"
JSVERSION_1_2 "1.2"
JSVERSION_1_3 "1.3"
JSVERSION_1_4 "1.4"
JSVERSION_ECMA_3 "ECMAv3"
JSVERSION_1_5 "1.5"
JSVERSION_1_6 "1.6"
JSVERSION_1_7 "1.7"
JSVERSION_1_8 "1.8"
JSVERSION_ECMA_5 "ECMAv5"
JSVERSION_DEFAULT "default"
Other NULL
