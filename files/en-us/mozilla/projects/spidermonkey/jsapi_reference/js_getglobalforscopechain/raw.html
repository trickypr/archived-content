Returns the global object for the active function on the context. Renamed to -->.

Syntax

JSObject *
JS_GetGlobalForScopeChain(JSContext *cx);
Name Type Description
cx --> The context for which to return the global object. -->

Description

JS_GetGlobalForScopeChain() returns the global object for whatever function is currently running on the context. In other words, it returns the global object on the current scope chain.

If there is no JavaScript running on the context, this returns the context's global, i.e., -->(cx). Note that JS_GetGlobalObject() and context globals are becoming obsolete.
