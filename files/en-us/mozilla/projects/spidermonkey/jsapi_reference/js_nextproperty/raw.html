Get the next property id from an iterator object produced by .

Syntax

bool
JS_NextProperty(JSContext *cx, JS::HandleObject iterobj, JS::MutableHandleId idp);
Name Type Description
cx The JS context in which to enumerate properties.
iterobj The opaque iterator object returned by a previous call to .
idp Out parameter. On success, *idp receives the name or index of the next property name; or JSID_VOID if there are no more enumerable properties to visit.

Description

On success, JS_NextProperty returns true, with *idp containing the id of the next enumerable own property to visit using iterobj, or JSID_VOID if there is no such property left to visit. On error, it returns false, and the value left in *idp is undefined.
