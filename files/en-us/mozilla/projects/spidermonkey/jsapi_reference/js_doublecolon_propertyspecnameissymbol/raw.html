Determine if the given or value is actually a symbol code and not a string.

Syntax

bool
JS::PropertySpecNameIsSymbol(const char *name);
Name Type Description
name const char * The pointer of the name, actually the uintptr_t type, and not a pointer to any string.

Description

JS::PropertySpecNameIsSymbol determines if the given or value is actually a symbol code and not a string, and returns true if so.
