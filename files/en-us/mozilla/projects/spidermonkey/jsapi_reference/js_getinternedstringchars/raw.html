Get the chars of an interned string.

Syntax

const jschar *
JS_GetInternedStringChars(JSString *str);

const jschar *
JS_GetInternedStringCharsAndLength(JSString *str, size_t *length);
Name Type Description
str The interned string.
length size_t * Out parameter. Receives the length of the string.

Description

JS_GetInternedStringChars returns a pointer to the interned string.

JS_GetInternedStringCharsAndLength returns a pointer to the interned string and stores the length of it to *length.
