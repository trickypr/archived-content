The callback is called whenever an object property is defined. It implements .

Syntax

JSBool (*JSDefinePropOp)(JSContext *cx, JSObject *obj, jsid id, jsval value,
    JSPropertyOp getter, JSPropertyOp setter, unsigned int attrs);
Name Type Description
cx Pointer to the JS context in which the property is being defined.
obj The object on which the property is being defined.
id The name or index of the property to define.
value The initial value for the new property.
getter The getter for the new property.
setter The setter for the new property.
attrs The for the new property.

Description

Define obj[id], an own property of obj named id, having the given initial value, with the specified getter, setter, and attributes.
