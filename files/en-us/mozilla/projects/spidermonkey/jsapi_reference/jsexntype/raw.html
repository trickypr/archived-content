Possible exception types.

Value Prototype in JavaScript
JSEXN_NONE An unthrowable error. (lower bound)
JSEXN_ERR -->
JSEXN_INTERNALERR -->
JSEXN_EVALERR -->
JSEXN_RANGEERR -->
JSEXN_REFERENCEERR -->
JSEXN_SYNTAXERR -->
JSEXN_TYPEERR -->
JSEXN_URIERR -->
JSEXN_LIMIT (upper bound)

Description

These types are part of a --> structure. They define which error to throw in case of a runtime error. JSEXN_NONE marks an unthrowable error.
