Cast a jsval to a JSObject * without a type check.

Syntax

JSObject *
JSVAL_TO_OBJECT(jsval v);

Description

JSVAL_TO_OBJECT casts the argument, v, to type JSObjectÂ *. As a precondition, (v) must be true. That is, v must be either a reference to an object or JSVAL_NULL. This function is not type-safe: if v is not an object or JSVAL_NULL, the behavior is undefined.

To convert a value to an object, use the function, which has well-defined behavior even when the argument is not an object or null.
