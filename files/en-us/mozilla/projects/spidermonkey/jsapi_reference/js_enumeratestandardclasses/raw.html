Enumerate standard class in an object.

Syntax

bool
JS_EnumerateStandardClasses(JSContext *cx, JS::HandleObject obj);
Name Type Description
cx Pointer to the executable script context for which to initialize JS function and object classes.
obj An object to enumerate standard classes.

Description

The global object's class's enumerate op should call JS_EnumerateStandardClasses(cx, obj), to define eagerly during loops any classes not yet resolved lazily.
