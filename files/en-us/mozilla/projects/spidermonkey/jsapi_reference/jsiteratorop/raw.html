Callback for creating iterators.

Syntax

typedef JSObject *
(*JSIteratorOp)(JSContext *cx, JSObject *obj, JSBool keysonly);
Name Type Description
cx Pointer to the JS context in which the iterator creation should take place.
obj Pointer to the object to iterate over.
keysonly If true, the iterator should yield keys only, not pairs.

Description

The JavaScript engine calls the .iteratorObject callback to create an iterator object for a given object. The callback should return an iterator object or NULL if an error or exception occurred on cx.
