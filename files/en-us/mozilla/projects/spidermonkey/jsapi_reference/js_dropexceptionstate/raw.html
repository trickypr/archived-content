Destroys a object previously created using .

Syntax

void
JS_DropExceptionState(JSContext *cx, JSExceptionState *state);
Name Type Description
cx Pointer to a JS context from which to derive runtime information.
state Pointer to the JSExceptionState object to destroy.

Description

This function destroys the specified object, unrooting as necessary any attached exception object and freeing the memory resources associated with the JSExceptionState object.

This object should previously have been created using .
