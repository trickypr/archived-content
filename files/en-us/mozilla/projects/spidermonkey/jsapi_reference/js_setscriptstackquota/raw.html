Set the maximum amount of memory a context will use for certain data structures.

Syntax

void
JS_SetScriptStackQuota(JSContext *cx, size_t quota);
Name Type Description
cx --> The context to configure.
quota size_t The desired stack quota setting, in bytes.

Description

Set the quota on the number of bytes that stack-like data structures can use when the runtime compiles and executes scripts. These structures also consume heap space, so --> does not bound their size. The default quota is 32MB which is quite generous.

The function must be called before any script compilation or execution API calls, i.e. either immediately after --> or from the --> context callback.
