Root an internal fixed-size array of s.

Syntax

JS::AutoValueArray<N> vp(cx);
Name Type Description
cx The context in which to add the root.
N size_t Size of a array.

Description

JS::AutoValueArray<N> holds a rooted array of . This is typically used for local variables being passed to function which requires or a pointer to array.

Examples

Pass to function as a pointer to .

bool
someFunction1(JSContext *cx, unsigned argc, const JS::Value *argv) {
  /* ... */
}

JS::AutoValueArray<2> args(cx);
/* ... Initialize args[0] and args[1] here ... */
someFunction1(cx, 2, args.begin());

Pass to function as .

bool
someFunction2(JSContext *cx, const JS::HandleValueArray &args) {
  /* ... */
}

JS::AutoValueArray<2> args(cx);
/* ... Initialize args[0] and args[1] here ... */
someFunction2(cx, args);
