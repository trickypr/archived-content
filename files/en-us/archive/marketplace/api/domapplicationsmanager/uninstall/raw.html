Obsolete
This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

Uninstalls an app from the user's repository.

Syntax

navigator.mozApps.mgmt.uninstall( origin, [onsuccess], [onerror] )

Parameters

origin
The origin of the app to be removed.
onsuccess
A callback function that is invoked after the app has been successfully uninstalled.
onerror
A callback function that is invoked if the specified app doesn't exist; it is passed an error object with a code of noSuchApp.
