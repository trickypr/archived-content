Summary

The BluetoothStatusChangedEvent API provides access to information regarding any change to the status of a Bluetooth device.

A status change occurs when one of the following events is triggered:

  • : It occurs when an A2DP connection status changes. See for more information.
  • : It occurs when an HFP connection status changes. See for more information.
  • : It occurs when the pairing status of the device changes. See for more information.
  • : It occurs when an SCO connection status changes. See for more information.

Interface overview

interface BluetoothStatusChangedEvent: Event
{
  readonly attribute DOMString address;
  readonly attribute boolean status;
};

Properties

A string representing the address of the device for which the status has changed in the Bluetooth micro-network.
A boolean representing the current status of the connection. It can be enabled (true) or disabled (false).

Methods

None.

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
