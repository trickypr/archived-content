The ::-ms-expand CSS pseudo-element is a Microsoft extension that represents the button of a menu control that opens or closes the drop-down menu. Typically it is a triangle that points downward.

Allowable Properties

Only the following CSS properties can be used in a rule with ::-ms-expand in its selector. Other properties are ignored.

  • (values block, inline-block, none)

Syntax

Specifications

Not part of any specification.

Browser compatibility

Historically supported in Internet Explorer and Edge before version 79.
