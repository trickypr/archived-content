The devicelight event is fired when fresh data is available from a light sensor.

Note: This event has been disabled by default in Firefox 62, behind the device.sensors.ambientLight.enabled preference ().

Bubbles No
Cancelable No
Interface SensorCallback
Target DefaultView (window)

Other properties

Property Type Description
value Double (float) The sensor data for ambient light in Lux.
min Double (float) The minimum value in the range the sensor detects (if available, 0 otherwise).
max Double (float) The maximum value in the range the sensor detects (if available, 0 otherwise).

Examples

window.addEventListener('devicelight', function(event) {
  console.log(event.value);
});
